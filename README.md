Allure-2 report example for Cypress
-----------------------------------
> This repo an example of how to use [cypress-allure-example](https://www.npmjs.com/package/@shelex/cypress-allure-plugin) plugin

#### Usage to run locally
- `$ yarn cy:run`
- `$ yarn allure:report`
- `$ allure open`
